## Format Preferences

### Hook Style
- [ ] Question that challenges assumptions
- [ ] Bold statement or hot take
- [ ] Story opener ("Last week I...")
- [ ] Surprising statistic or fact
- [ ] Direct address ("You're doing X wrong")

### Length
- [ ] Short (under 500 characters) - punchy, single idea
- [ ] Medium (500-1000 characters) - main point + context
- [ ] Long (1000-1500 characters) - story format with lessons
- [ ] Varies by topic

### Line Breaks
- [ ] Single sentence per line (scannable)
- [ ] Short paragraphs (2-3 sentences)
- [ ] Mix depending on rhythm

### Emoji Usage
- [ ] None - purely text
- [ ] Minimal - 1-2 for emphasis
- [ ] Moderate - section headers or bullets
- [ ] Liberal - expressive throughout

### Hashtags
- [ ] None
- [ ] 1-3 relevant hashtags at end
- [ ] 3-5 hashtags
- [ ] Integrated into text
